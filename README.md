# 💰 MoneyPal

MoneyPal is a Flutter-based expense splitting application that helps users manage shared expenses, track balances, and settle payments easily within groups.

---

## 🚀 Features

* 👥 Create and manage groups
* 💸 Add and split expenses among members
* 📊 Real-time balance tracking
* 🔄 View transaction history
* 🔐 Firebase Authentication
* ☁️ Cloud Firestore integration

---

## 🛠️ Tech Stack

* Flutter (Dart)
* Firebase Authentication
* Cloud Firestore
* Material UI

---

## 📱 How It Works

1. Create a group
2. Add members
3. Record expenses
4. Automatically calculate who owes whom
5. Settle balances easily

---

## ⚙️ Installation

```bash
git clone https://github.com/urva2512/MoneyPal.git
cd MoneyPal
flutter pub get
flutter run
```

---

## 🔐 Firebase Setup

1. Create a Firebase project
2. Add Android app
3. Download `google-services.json`
4. Place it inside:

```
android/app/
```
